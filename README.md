# **Эксклюзионная хроматография**

**Эксклюзионная хроматография** --- метод хроматографии, при котором  
молекулы в растворе разделяются по размеру. Образец вносится в колонку,  
заполненную гелем или пористыми частицами неподвижной фазы. Разделение  
происходит за счёт многократного обмена молекул компонентов образца  
между подвижной фазой, протекающей между частицами, и неподвижной фазой,  
находящейся в порах.  

Метод эксклюзионной хроматографии используют для вычисления  
среднечисловых и среднемассовых молекулярных масс полимеров, а также  
получения данных для построения кривых молекулярно-массового  
распределения.  

Программа моделирует работу с жидкостным хроматографом и создана для  
лабораторного практикума по дисциплине «Высокомолекулярные соединения»  
для студентов, обучающихся на специальностях «Фундаментальная и  
прикладная химия» и «Химия».  

Ранее при проведении лабораторных занятий использовалась программа 2011  
года, написанная на Objective Pascal (среда Delfi). По прошествии  
четырнадцати лет появился ряд проблем, приводящих к невозможности  
использования данной программы на современных операционных системах. При  
попытке запуска прибора построение графика не производилось, что связано  
с некорректной работой макросов, заключенных в excel-файле.  

Обновленная версия программы встречает красочным цвета интерфейса,  
создающим настроение при работе с ней.

> **Возможности Приборной панели:**  

- По клику в поле «*Доступные образцы*» для выбора представлен  
  определенный набор полимеров: стандартные образцы с известными  
  молекулярными массами, использующиеся студентами для построения  
  градуировочного графика, и исследуемые образцы, требующие проведения  
  анализа и полного исследования;  

- При нажатии кнопки «*Загрузить образец*» наименование выбранного  
  полимера выносится в отдельное поле. Данный этап отвечает за  
  готовность полимера к предварительной пробоподготовке;  

- Кнопка «*Подготовить образец*» имитирует загрузку пробы полимера в  
  колонку жидкостного хроматографа. Так как в условиях химической  
  лаборатории данный процесс не мгновенен по времени, реализована  
  динамическая загрузка образца (поле во времени заполняется цветовой  
  заливкой);  

- При нажатии «*Запуск прибора*» осуществляется построение  
  хроматограммы в левой части главного окна программы. Внешний вид  
  кривой соответствует работе самописца: на график добавлены шумы и  
  сдвиги;  

- Для сохранения четкой последовательности действий и формирования  
  образа работы с хроматографом, доступ к последующим этапам работы  
  предоставляется по очередности: невозможно подготовить образец без  
  его загрузки или запустить «пустой» прибор;  

- Поскольку основной деятельностью студентов в рамках данной  
  лабораторной работы является обработка полученных хроматограмм, для  
  удобства реализована возможность печати: в зоне печати отображаются  
  наименование полимера, график с подписанными осями и указанными  
  размерностями. Для выбора необходимой области печати график можно  
  подвинуть правой кнопкой мыши. Процесс печати осуществляется  
  автоматически в необходимой альбомной ориентации листа, что  
  минимизирует самостоятельную деятельность студентов в технических  
  аспектах;  

- Оптимально подобранная ширина площади под кривой позволяет с  
  легкостью осуществлять построение требуемого числа высот в области  
  графика;  

- При нажатии кнопки «Сброс» происходит «очистка» хроматографа от  
  выбранной пробы и возможность выбора другого образца для дальнейшей  
  работы.  

Уникальной возможностью обновленный программы является добавление  
преподавателем новых образцов для исследования. Таким образом,  
модернизация данной лабораторной работы осуществима не только в области  
совершенствования программного пакета, но и рассмотрения разнообразного  
набора полимеров с различной степенью полидисперсности и  
молекулярно-массовыми характеристиками. Для корректной работы требуется  
создать новый лист и заполнить его данными по шаблону предыдущих вкладок  
excel-файла.
